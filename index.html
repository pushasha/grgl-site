What is GRGL?

Pronounced "Gurgle", GRGL is a highly extensible and easy-to-use API/library to simplify the management of common RPG elements such as quests, dialogue, characters, etc. in the development of role-playing video games. This version of GRGL is meant to be used with the C# .NET programming language.

Designed for developers

Every method and class is documented with IntelliSense-compatible XML comments; Follows MSDN C# conventions for easy-to-read code.

Time-saver

GRGL keeps you from re-inventing the wheel — commonly-used game elements and systems are provided as concrete classes and interfaces.

Game Engine Integration

Written in C# .NET, allowing for easy integration into popular game engines such as Unity, Unreal Engine, and Microsoft XNA.

Debug-Friendly

GRGL is structured for easy unit-testing and debugging. No more time wasted on complex unit-test writing and re-writing.

Easy to customize

Systems are abstracted at every level of complexity, making them easy to extend and customize.

Free and Open-Source

GRGL is free to use, and open-source under the GNU Lesser General Public License.

Features

  • Free and Open-Source
  • Visual C# .NET
  • No third-party dependencies
  • Fully documented with XML comments
  • Easily customizable and extensible
  • Applies to almost all types of games, not just RPGs
  • Structured for easy debugging and testing
